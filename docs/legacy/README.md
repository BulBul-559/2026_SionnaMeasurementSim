# Legacy Docs

`docs/legacy/` 用于临时存放已经过时、但暂时不确定是否要删除的文档。开发过程中如果发现
文档内容明显不再代表当前系统事实，可以把它移动到这里，并在提交说明中写明原因。

约定：

- `docs/sys/` 和 `docs/agent_handoff.md` 永远代表当前系统事实。
- `docs/todo/` 代表当前 active TODO 和完成历史。
- `docs/performance/` 是历史实验记录；只有明显误导当前事实、且不再适合作为实验记录保留的文档，
  才移动到本目录。
- 本目录内容由人工定期复核，决定保留、删除或重新整理。
